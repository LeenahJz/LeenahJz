Esta estructura HTML detalla la página web para un hospital, consta de una sección inicial, reseñas, presentación del equipo médico, servicios ofrecidos y el formulario de contacto para dejar a la página. 

[Ejercicio 2, Módulo2] En esta ocasión se implementa la modularización con SASS, metodología BEM y el media queries.

------Explicación de web-------
Inicio
-Da la bienvenida a la página, indica la misión y visión del hospital. Cada uno dividido en cards por BEM. También se aplicó media queries en cada apartado.

La semántica se compone como HTML5, dividiendo cada módulo por header, nav, footer y otros más básicos como variables globales y reseteo cuando se recargue la página

Equipo Médico, con section se diseña todo este apartado y se crea clases para modificar títulos, textos y contenidos.
-Presenta a los profesionales que trabajan en el hospital
Contacto

Contacto, con section se diseña todo este apartado y se crea clases para modificar títulos, textos y contenidos.
-Deja un formulario para los usuarios que quieran ser contactados por el hospital.
Además, se aplica media queries para las imagenes y el mapa para que estos se acomoden al tamaño de la pantalla.
