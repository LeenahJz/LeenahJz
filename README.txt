Esta estructura HTML detalla la página web para un hospital, consta de una sección inicial, reseñas, presentación del equipo médico, servicios ofrecidos y el formulario de contacto para dejar a la página. 

[Ejercicio 3, Módulo2] En esta ocasión se implementa la metodología BEM y el framework 
Bootstrap para mejorar la estructura visual.

------Explicación de web-------
Inicio
-Da la bienvenida a la página, indica la misión y visión del hospital. Cada uno dividido en 
cards por BEM, utilizando grilla se dividio la página en filas y columnas. También se aplicó 
media queries en cada apartado.

La semántica se compone como HTML5, dividiendo cada módulo por header, nav, footer y otros 
más básicos como variables globales y reseteo cuando se recargue la página

Cuando se presiona 'Equipo Médico', el link te dirige a esa sección.
-Presenta a los profesionales que trabajan en el hospital
Contacto
-Deja un formulario adaptado de bootstrap y personalizado para los usuarios que quieran ser contactados por el hospital.
